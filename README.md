# Alura SPA

Esse projeto consiste em um site de um SPA fictício chamado Alura SPA. O objetivo do site é apresentar os serviços oferecidos pelo SPA, permitir que os usuários se inscrevam para receber descontos, e apresentar produtos disponíveis para compra, e o meu objetivo é aplicar o que eu aprendi estudando SASS.

## Tecnologias utilizadas ⚒ 

- HTML
- SASS

## Instalação

1. Clone o repositório: `git clone https://github.com/darkmoonsk/sass-alura-spa.git`
2. Abra o arquivo `index.html` no seu navegador de preferência.

## Funcionalidades ⚙️

### Serviços

Na seção "O que oferecemos", são apresentados os serviços oferecidos pelo Alura SPA. Cada serviço é apresentado em um card, contendo uma imagem ilustrativa, título e descrição.

### Inscrição para descontos

Na seção "Inscreva-se para ganhar descontos!", os usuários podem preencher um formulário para se inscrever e receber descontos.

### Produtos à venda

Na seção "É sobre bem estar", são apresentados produtos disponíveis para compra, como sabonetes, sais de banho e velas.
